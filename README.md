# minkiele.github.io

## Instructions to build the project

1. Checkout this repository
2. Run `npm install`
3. Run `npx nx run public:build` to build the project and generate the public files
4. Host it somewhere.

## What about env vars?

There are some environment variables you must set to have the website up and running, and their definitions are in [`src/env.d.ts`](./src/env.d.ts).

## Where is the source code?

All the examples are located inside the `packages/public/src/apps` directory.

## What now?

Have fun.
